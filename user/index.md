# ユーザーデータの通信（API）

ユーザーデータの通信にまつわる解析内容をまとめる。

デレステでは http://game.starlight-stage.jp/ にあるエンドポイントをPOSTで叩くことで、ユーザー個別の通信を行っている。

ユーザーデータのサンプルは、実際の通信をもとに、新たに生成した偽のユーザーデータであることに留意されたい。
