# エンドポイント一覧

[概要](general.md) でも説明したとおり、ホスト名は http://game.starlight-stage.jp/ である。

たとえば VersionCheck であれば http://game.starlight-stage.jp/load/check になる。

これは `ApiType` に記載されている情報の焼き直しである。

このリストはバージョンアップによって増減する可能性があるため注意する必要がある。
